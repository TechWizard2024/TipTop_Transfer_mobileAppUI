Chaque ligne du fichier excel Feuil4-Unified_Build contient (dans les colonnes ci-dessous) la description d'un seul UI :
UI_ID : ID du UI
UI_Name : Nom du UI
UI_img_name : Nom de l'image de l'UI
My_Observation_On_Deepseek_Processing_Compared_MyView_On_UI_img_name

User_Target : Nom du Type de profil utilisateur (userProfilType) cible
Target_App : Nom de l'App (TipTop Transfer ou TipTop Transfer Business ou TipTop Transfer Agent) où le UI_img_name correspondant, est applicable
UI_Description : Description basique de l'image UI_img_name par moi-meme
App_Config_variable : Variable ou donnée de configuration generale (app) basique pertinent pour l'UI de l'App
AI-Raw_UI_Description-Flutter : Description du Screenshot de UI UI_img_URL exactement comme Deepseek voit le UI pour reconscruire ce UI dans un contexte flutter
AI-Raw_App_Config_variable-Flutter : Description de variables de configuration generale (dans les categories app, server, utilisateur) à partir de AI-Raw_UI_Description-Flutter
AI-Raw_UI_Description-EyeView : Description du Screenshot de UI UI_img_URL exactement comme Deepseek voit le UI (EyeView)
AI-Raw_App_Config_variable-EyeView : Description de variables de configuration generale (dans les categories app, server, utilisateur) à partir de AI-Raw_UI_Description-EyeView
AI-Raw_UI_Description-UnifiedView
AI-Raw_App_Config_variable-UnifiedView
UI_Modification_Requested : Modifications à apporter à UI_img_URL
UI_General_Modification_Requested-Modification_App_wide : Modifications à apporter à l'echelle de l'App
Potential_Compatibility_Issues_for_Device_with_Old_OS ( Android >= 10 ; iOS>= 12 ) : Liste (par tiret -) de Problèmes potentiels de compatibilité pour les smartphones fonctionnant sous d'anciens systèmes d'exploitation (Android >= 10 ; iOS >= 12)
Solutions_for_Potential_Compatibility_Issues_for_Device_with_Old_OS ( Android >= 10 ; iOS>= 12 ) :  Liste (par tiret -) de Solution aux Problèmes potentiels de compatibilité pour les smartphones fonctionnant sous d'anciens systèmes d'exploitation (Android >= 10 ; iOS >= 12)
