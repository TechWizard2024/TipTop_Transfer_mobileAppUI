# Application_mobile_TipTop_Transfer: Frontend utilisateur, simple et rapide

	## UI description : 
		3 Apps ou Frontend differents contecté à Serveur_Cloud_Backend via internet
		- App1 or Frontend1:
			App Name : TipTop Transfer
			Targeted User : Customer
			UI exemple : https://github.com/TechWizard2024/TipTop_Transfer_mobileAppUI/Screenshot Wave
		- App2 or Frontend2:
			App Name : TipTop Transfer Business
			Targeted User : Business
			UI exemple : https://github.com/TechWizard2024/TipTop_Transfer_mobileAppUI/Screenshot Wave Business
		- App3 or Frontend3:
			App Name : TipTop Transfer Agent
			Targeted User : Agent
			UI exemple : https://github.com/TechWizard2024/TipTop_Transfer_mobileAppUI/Screenshot Wave Agent
		- UI description detail : https://github.com/TechWizard2024/TipTop_Transfer_mobileAppUI/blob/main/App_Wave_UI_List.xlsx 
			App_Wave_UI_List.xlsx contient les colonnes ci-dessous : 
			UI_ID : ID du UI
			UI_Name : Nom du UI
			UI_img_name : Nom de l'image de l'UI
			UI_Folder_name : Nom du dossier contenant l'image de l'UI
			User_Target : Nom du Type de profil utilisateur (userProfilType) cible
			Target_App : Nom de l'App (TipTop Transfer ou TipTop Transfer Business ou TipTop Transfer Agent) où le UI_img_name correspondant, est applicable
			UI_Description : Description basique de l'image UI_img_name
			App_Config_variable : Variable ou donnée de configuration generale (app) basique pertinent pour l'UI de l'App
			UI_Modification_Requested : Modifications à apporter à UI_img_name
			UI_General_Modification_Requested_[Modification_App_wide] : Modifications à apporter à l'echelle de l'App
			Comment : Commentaire descriptif additionel de UI_img_name correspondant
